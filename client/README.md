# Đồ án môn *Phát triển các Hệ thống thông minh*, đề tài *Website đánh cờ vua*

## Project Setup

```sh
npm install
```

### Chế độ phát triển với hot reload

```sh
npm run dev
```

### Biên dịch và đóng gói thành sản phẩm

```sh
npm run build
```

### Kiểm tra sơ bộ với [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
